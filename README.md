# Keyboard Checker

キーボードの全キーが正しく動作するかテストするWebアプリ（PWA）。新しいキーボードの購入時や、キーの故障を確認したいときに使えます。

## 機能

- **105キー対応**: フルサイズキーボード（ANSI配列）の全キーをカバー
- **リアルタイム検出**: キーを押すと即座にハイライト表示
- **カバレッジ表示**: テスト済みキー数と進捗率をプログレスバーで表示
- **キー情報表示**: 押したキーの `key`, `code`, `keyCode`, `location` をリアルタイム表示
- **未登録キー検出**: レイアウトに含まれないキーが押された場合も記録
- **PWA対応**: インストールしてオフラインでも動作

## 使い方

1. ブラウザでページを開く（GitHub Pages等でホスト）
2. キーボードのキーを1つずつ押していく
3. テスト済みのキーは緑色にハイライトされる
4. 全キーのテストが完了するとカバレッジが100%になる
5. 「Reset」ボタンでテスト状態をクリア

PWA対応のため、ブラウザからインストールすればオフラインでも利用可能です。

## ファイル構成

```
keyboard-checker/
├── index.html      # HTML構造（キーボードレイアウト）
├── style.css       # スタイル（ダークテーマ）
├── app.js          # キー検出ロジック
├── manifest.json   # PWAマニフェスト
├── sw.js           # Service Worker（オフライン対応）
└── README.md
```

## 技術的な補足

- フレームワーク不使用。HTML + CSS + Vanilla JS のみ
- キーボードレイアウトは CSS Flexbox + Grid で構成
- 一部ブラウザ/環境で `e.code` が空になる修飾キー（右Shiftなど）は `e.key` + `e.location` から推定
